You’re in Bandit20. Running “ls” will help you see “suconnect”, which is a segued (highlighted in red). Running it will tell you what you can already see in the question itself (that you need a port number), but a new piece of info is that it’s a TCP connection. For that, you need “nc” and checking the manual will help you see “-l” which is for listening. Write a port number of your choice.
Once you’re in, open another terminal and run the setuid with “./suconnect” which will wait for you to enter the Bandit20 password from the “nc” command. This will give you the next password.
