#Bandit Level 0 - Get inside the lab

Use the “ssh command”. They tell you port 2220, so use “-p 2220”. 
The synopsis shows you also need a destination, which is bandit0@bandit.labs.overthewire.org. 
Full command is “ssh -p 2220 bandit0@bandit.labs.overthewire.org” and you introduce “bandit0” when asked for the password.
