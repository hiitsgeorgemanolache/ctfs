Similar to the previous level, if you look in the “find” manual, you will see “-group” and ”-user”, so the command is “find -user bandit7 -group bandit6 -size 33c”. This will give you a lot of denied permissions, but also “./var/lib/dpkg/info/bandit7.password” which you can then “cat”. After extra investigation, if you add “2>/dev/null” to the command, it won’t print any denied permission.
